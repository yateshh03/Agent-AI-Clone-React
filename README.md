# 🤖 Agent AI Clone (React + Tailwind)

A modern and responsive **Agent AI Clone UI** built using **React** and **Tailwind CSS** with full support for **Dark & Light Theme**.

👨‍💻 Author: **Yatesh Sharma**

🔗 Live Demo: https://playful-naiad-497171.netlify.app/ 

---

## 🚀 Features

- ⚡ Fully Responsive (Mobile + Tablet + Desktop)
- 🌙 Dark & Light Mode Toggle
- 🎨 Clean & Modern UI Design
- 💻 Built with React + Tailwind CSS
- 🔥 Smooth user experience
- 📱 Mobile-first approach
